Introduction to HTML

HTML (HyperText Markup Language) is the standard language used to create web pages.

It provides the structure of a webpage and is the foundation of web development.

  • Defines the structure of web content
  • Supported by all web browsers
  • Easy to learn and essential for web development
HTML Structure

An HTML document is structured with elements that describe the content and layout.

The basic structure includes:




  
    Page Title
  
  
    
Hello World

    
This is an HTML document.
Elements and Tags

HTML is composed of elements, each represented by tags.

Tags are surrounded by angle brackets, like <p> for paragraphs.

Example of a header tag:

<h1>This is a header</h1>
Attributes

Attributes provide additional information about elements. Common attributes include:

  • class: Specifies a class name for styling
  • id: Provides a unique identifier for an element
  • src: Specifies the source of an image or other media
  • href: Defines a link destination in anchor tags

Example of an image with attributes:

<img src="image.jpg" alt="Sample Image" width="200">
Semantic Elements

Semantic elements describe the purpose of content, improving readability and accessibility.

  • <header>: Represents a header for a section
  • <article>: Contains self-contained content
  • <section>: Defines a section in a document
  • <footer>: Represents the footer of a document or section

Example of semantic elements in HTML:
